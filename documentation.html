Overview

Intellect Library is a static promotional website for a study library located in Kahrai, Agra, Uttar Pradesh. It is a fully front-end project — no server, no build tool, no framework. Every page is a single index.html file styled with one CSS file and powered by one vanilla JS file.

Business: Intellect Library, Kahrai, Shamsabad Road, Agra, UP  |  Phone: +91 79835 18266  |  Hours: 8 AM – 8 PM, 7 days

JavaScript Features

  • Navbar scroll effect — adds .scrolled class after 80 px of scroll
  • Mobile hamburger toggle — CSS-animated ☰ → ✕ transition using span transforms
  • Click-outside-to-close mobile nav
  • Active-section highlight in nav using IntersectionObserver
  • Scroll-reveal animations (.reveal → .visible)
  • Counter animation for hero stats on viewport entry
  • Gallery lightbox (pure JS DOM overlay, no library)
  • Smooth scroll for all #anchor links
  • Back-to-top button

Responsive Design

No CSS framework. All responsive behaviour is written with hand-crafted media queries.

BreakpointTargetKey Changes
max-width: 900px Tablets Hamburger menu shown; plans grid → 2 columns
max-width: 680px Portrait phones Nav tagline hidden; hero padding adjusted; single-column gallery & plans; footer stacked
max-width: 480px Small phones Further font & spacing reductions
orientation: landscape
max-height: 520px		 Landscape phones Compact navbar; hero height auto with reduced padding; 2-col gallery & plans; scroll indicator hidden

Mobile Nav Behaviour

On screens ≤900 px wide (or landscape phones), the nav links slide in from the right as a 260 px fixed drawer (right: -100% → right: 0). The hamburger icon animates to an ✕ via CSS transform. The drawer closes on:

  • clicking a nav link
  • clicking anywhere outside the drawer
  • clicking the ✕ hamburger again

SEO & Meta

Head Tags

  • Primary SEO: <title>, description, keywords, robots, canonical URL
  • Geo / Local SEO: geo.region, geo.placename, geo.position, ICBM (lat/lng 27.138, 78.044)
  • Open Graph: og:type, og:url, og:title, og:description, og:image (1600×1067)
  • Twitter Card: summary_large_image

Structured Data (JSON-LD)

A LocalBusiness schema is embedded in <script type="application/ld+json"> covering name, address, geo coordinates, telephone, opening hours, price range, and social profiles.

Performance Optimisations

  • <link rel="preconnect"> for Google Fonts domains
  • <link rel="preload" as="image"> for above-fold gallery image
  • Font Awesome loaded from Cloudflare CDN (cached globally)

Local Development

No install or build step needed. Serve the root folder with any static file server.

Python (recommended — pre-installed on most systems)

# Python 3
python -m http.server 8000

# Python 2 (legacy)
python -m SimpleHTTPServer 8000

Then open http://localhost:8000 in a browser.

Node.js alternatives

# One-liner with npx (no install)
npx serve .

# Or install globally
npm install -g serve
serve .

VS Code

Install the Live Server extension and click Go Live in the status bar.

Deployment

The site is deployed to two platforms simultaneously from the same main branch.

GitHub Pages
Auto-deploys on push to main
https://jaichaturvedi88.github.io/intellect-library/
  • Configured in: Settings → Pages → Deploy from branch → main / (root)
  • .nojekyll file prevents GitHub's Jekyll processor from interfering
  • No CLI needed — push triggers deploy automatically
Netlify
Manual deploy via Netlify CLI
https://intellectlibrary.netlify.app
  • Site name: intellectlibrary
  • Account: jai.chaturvedi88@gmail.com
  • No build command — --no-build flag used
  • Publish directory: . (project root)

Netlify Deploy Command

netlify deploy --dir . --prod --site intellectlibrary --no-build

Netlify CLI Setup (one-time)

# Install Netlify CLI globally
npm install -g netlify-cli

# Authenticate (opens browser)
netlify login

Custom Domain

The canonical URL in HTML and JSON-LD is set to https://www.intellectlibrary.in/. When a custom domain is connected, update the Netlify domain settings and verify DNS records.
